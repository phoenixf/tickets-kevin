Gostaria de um sistema em php, para abertura de tickets.
A abertura é feita a partir de um modal com os campos para preencher.

Após abrir um ticket, vai para a lista.

onde existe uma lista de tickets, que lista todos os tickets abertos.
Na listagem de ticket, terá os campos: ID ticket, Usuario, data de abertura, responsável, prioridade, situação atual

As prioridades são: baixa, normal, alta, urgente, PRIORIDADE.

Na tabela de tickets, gostaria que fizesse uma separação por cor, onde cada tipo de prioridade tem uma cor determinada
